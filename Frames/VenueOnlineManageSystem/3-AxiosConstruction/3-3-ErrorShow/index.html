# 错误展示

TIP

  1. 错误拦截统一使用错误码拦截，错误码在Error中的Message字段中
  2. 错误展示使用错误码对应的错误弹框进行展示
  3. 错误拦截有vant框架自带的Toast插件与自己封装的MessageDialog插件进行展示，MessageDialog插件提供了$props与$event接口，与使用普通的组件并无不同
  // MessageDialog插件
  MessageDialog.$create({
    $props: message,
    $events: {
      click() {
        jumpToUpdatePage()
      }
    }
  }).show()

  // Toast插件
  Toast(error.message)
